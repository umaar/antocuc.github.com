Gpx Splitter

Gpx Splitter is a simple utility program that aids in separating large GPX files in multiple files each of which has a predetermined number of instructions.

Gpx Splitter is written in Java and is available for download for the major operating systems as a cross platform runnable jar file, windows specific executable (.exe) and mac specific application (.app). It is distributed for free under the Creative Commons 3.0.

Take a look at some screenshots of the application and see the documentation for a list of typical usages.

Gpx Splitter is currently not deployed with an installation script. The downloaded application can be directly executed by double-clicking on the file. Manual installation is possible and uncomplicated.

Documentation

Gpx Splitter was built with usability in mind. The main aim was to be able to receive output after the fewest number of clicks. The whole process consists of:
  1. Loading the gpx file to be split via the "Browse..." button
  2. Setting the number of instructions that we require each output file to have
  3. Setting the type of gpx we require as output let that be a "Track" or a "Route"
  4. Enter the name of the output files and click on "Save"
Gpx Splitter can be used to:
  • Split GPX files with a large number of instructions into multiple files with a smaller number of instructions.
  • Convert GPX tracks into GPX routes.
  • Convert GPX routes into GPX tracks.
Technical documentation of Gpx Splitter is available in browsable JavaDoc format, code behaviour can be seen via the JUnit tests and UML diagrams (class + use case) can be found on the GitHub repository.
